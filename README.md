# Chess Tic Tac Toe

A small chess-inspired game designed to help children learn chess piece movement.

Players use three pieces:

- Rook
- Bishop
- Knight

The game is played on a **3×3 board**.

## Rules

### Phase 1 – Placement

Players take turns placing pieces until each player has placed:

- 1 rook
- 1 bishop
- 1 knight

### Phase 2 – Movement

After placement, pieces move according to **standard chess movement rules**.

- Rook: horizontal / vertical
- Bishop: diagonal
- Knight: L-shape

There is **no capture**.

Pieces can only move to **empty squares**.

### Winning

A player wins by forming:

- 3 in a row
- 3 in a column
- 3 in a diagonal

## Purpose

This game was designed as a **learning tool for children**.
It helps train:

- piece movement
- pattern recognition
- tactical thinking

Games typically last **2–5 minutes**.
